文件提交需要手动确认的解决办法

在教师端右上角小齿轮设置中 → 接受文件选项 → 提交文件前需要老师确认 ，把这个取消勾选就好了，默认是勾选状态。

文件提交目录快速打开方法

在教室端在左侧导航栏选择 文件提交视图 → 在空白处右击鼠标 → 打开文件保存文件夹

这个文件夹也可以在设置中自己设置保存目录。

恶意频繁举手问题

这个很好解决 ，控制端主页面 有一个阻止举手的按钮。

恶意给教师端发无用消息

在控制端右侧 → 消息

建议设置内容

前言

在过去一段时间上机操作期间我发现的问题，以及我个人推荐设置的几项设置，我觉得可以有效抑制学生的小动作。 -skycola

在屏幕演示期间快速切换方向使软件卡退解决方案

在上机时不少发现总有一些同学不老实，利用电脑性能不足以及软件bug，将被控端（学生端）软件卡退，以达到退出控制的目的。

其中基本上他们都采用了一种方法：在教师发起屏幕演示期间通过快速切换屏幕方向让软件卡退。

解决方法

教师通过在教师端设置工具条的显示来达到让被控端无法切换屏幕方向的目的。

教师控制端右上角齿轮（设置） → 屏幕显示广播 → 禁止学生在屏幕广播时使用工具条。

注意：具体演示我录制视频了，可以来找我要，因为我的服务器空间带宽太低、延迟高，而我的图床只可以存储图片，不支持视频，所以我就没有放在这里。

极域课堂软件漏洞

在我实际测试中有一些漏洞，但是一般学生也不会，了解一下就好啦。

注册表明文存储极域课堂设置密码

HKEY_LOCAL_MAXHINE文件夹 → SOFTWARE 文件夹 → WOW6432Node文件夹 → TopDomain文件夹 → e-Learning Class Standard文件夹 → 最后在1.00文件夹下的，UninstallPassword里的便是密码了。

参考CDSN博文：https://blog.csdn.net/Superhero_tade/article/details/121694508

有这个密码有啥用呢？

学生可以切换连接频道以便于与控制端断开连接，是在被控端消失的那种，这个密码也用于卸载极域课堂students端软件。
